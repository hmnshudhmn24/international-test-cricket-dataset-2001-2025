# International Test Cricket Dataset (2001–2025)

A detailed ball-by-ball dataset covering **888 Test matches** played between **2001 and 2025**.
This dataset includes delivery-level events, player actions, innings data, and full match metadata—ideal for cricket analytics, research, and machine learning projects.

## Features
- 888 Test matches
- Ball-by-ball delivery data
- Runs, wickets, extras, dismissals
- Batting, bowling, and fielding details
- Match info: teams, venue, dates, innings, toss, result

## Format
Available in **JSON** and **YAML** formats.
